Q.60 Split the string into an array using split() Method?

Program :


    
The split method splits a string into an array of strings. It takes an argument for the
        delimiter, which can be a character to use to break up the string or a regular expression.
        For example, if the delimiter is a space, you get an array of words, and if the delimiter is
        an empty string, you get an array of each character in the string.

        Ex: Here we use split one string by spaces, then another by digits using a regular
        expression:

        <script>
        // Example string
            var myString = "Hello,World,How,Are,You";

            // Split the string into an array using the comma as a separator
            var myArray = myString.split(',');

            // Display the resulting array
            document.write(myArray);
        </script>
