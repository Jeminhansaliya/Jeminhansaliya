Q.61 Check if the string contains a word Script using includes() method?

Program :

    
In JavaScript, you can use the includes() method to check if a string contains a specific
        substring. Here's an example to check if the string contains the word "Script":


        <script>
        // Example string
            var myString = "This is a JavaScript Script example";

            // Check if the string contains the word "Script"
            var containsScript = myString.includes("Script");

            // Display the result
            document.write(containsScript);
        </script>
