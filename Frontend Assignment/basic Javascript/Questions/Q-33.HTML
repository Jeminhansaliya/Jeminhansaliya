33. Write to print Fibonacci series up to given numbers?

Program :


                <h3 id="ans"></h3>
    
                <script>
            
                    let a = 0, b = 1;
                    for (let i = 0; i <= 10; i++) {
                    let c = a + b;
                    document.getElementById("ans").innerHTML = c;
                    a = b;
                    b = c;
                    }
                </script>
