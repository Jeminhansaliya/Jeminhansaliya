Q-21)What is the use of Void (0)?

You might have occasionally came across “javascript:void(0)” in an HTML Document.

It is often used when inserting an expression in a web page might produce some unwanted effect.

To remove this effect, “javascript:void(0)” is used. This expression returns undefined primitive value.

This is often used with hyperlinks. Sometimes, you will decide to call some JavaScript from inside a link.
