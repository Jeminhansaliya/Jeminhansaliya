CSS

Cascading Style Sheets

Cascading Style Sheets is a style sheet language used for escribing the presentation of a document written in a markup language such as HTML or XML.

Basically CSS gives the outer cover on any HTML elements. If you consider HTML as a skeleton of the web-page then the CSS will be the skin of the skeleton.

way to adopt CSS

  1. Inline [Within tag with style attribute never recommaned]
  2. Internal [within page with style tag not preferabble]
  3. External [with .css extension file and linked with html page by link tag most used and recommned by W3C]
